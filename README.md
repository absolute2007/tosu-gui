# tosu-gui

Десктопный GUI для [tosu](https://github.com/tosuapp/tosu) — memory reader для osu! с in-game оверлеем.

Обычное окно приложения, а не вкладка браузера: статус, счётчики, оверлей, настройки, обновление tosu.

**Сайт:** [absolute2007.github.io/tosu-gui](https://absolute2007.github.io/tosu-gui/)

## Скачать

Windows 10/11 (x64), osu! Stable или Lazer.

1. Открой [сайт](https://absolute2007.github.io/tosu-gui/) или [releases](https://github.com/absolute2007/tosu-gui/releases/latest)
2. Поставь setup `.exe` или распакуй `.zip`
3. Запусти **tosu GUI**

`tosu.exe` уже лежит внутри сборки, отдельно ставить не нужно.

## Возможности

- статус tosu / подключение osu!
- установленные счётчики и каталог для скачивания
- настройки оверлея (горячая клавиша, FPS, AA)
- обновление tosu и самой tosu GUI из интерфейса
- настройка: по крестику — закрыть или свернуть в трей

## Разработка

```bash
npm install
npm start
```

`npm start` пересобирает UI/electron и запускает приложение.

Полный инсталлятор:

```bash
npm run build
```

Артефакты окажутся в `release/`.

## Мелочи

- Крестик по умолчанию закрывает приложение. «Сворачивать в трей» — в **Настройки**.
- Если после `npm start` открывается странное старое окно — закрой установленный tosu GUI и запусти снова (разные инстансы могли мешать друг другу).
